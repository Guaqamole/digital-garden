## 입력

첫째 줄에 정수 $n$, $m$ ( 0≤ $m$ ≤$n$ ≤ 2,000,000,000$0 \le m \le n \le 2,000,000,000$, $n \ne 0$)이 들어온다.

## 출력

첫째 줄에 $n \choose m$의 끝자리 0의 개수를 출력한다.

## 예제 입력 1 

```
25 12
```

## 예제 출력 1 


# 코드

```python
def solution(s):
    answer = [0, 0]

    def getBinaryNumToStr(num):
        return bin(num)[2:]

    while s != '1':
        zeros = s.count("0")
        tmp = s.replace("0", "")
        n = len(tmp)

        s = getBinaryNumToStr(n)
        answer[0] += 1
        answer[1] += zeros

    return answer
```

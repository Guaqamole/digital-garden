## 개발 필수요건

- API에서 회원정보는 '회원번호(회원에게 부여된 유니크한 번호)' 이외의 다른 정보는 전달받지 않음
- 각 API 에서 ‘회원번호’ 이외에 request, response는 자유롭게 구성
- 회원 별 적립금 합계는 마이너스가 될 수 없음
- 개발해야 할 API는 다음과 같습니다.
    - 회원별 적립금 합계 조회
    - 회원별 적립금 적립/사용 내역 조회
        - 페이징 처리
    - 회원별 적립금 적립
    - 회원별 적립금 사용
        - 적립금 사용시 우선순위는 먼저 적립된 순서로 사용(FIFO)
- ORM 사용 (ex: JPA / typeorm 등)
